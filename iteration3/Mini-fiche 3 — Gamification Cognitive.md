# Mini-fiche 3 — Gamification Cognitive

## Nom de la solution

**Gamification Cognitive** – Défis interactifs basés sur la sémantique.

## Problème ciblé

Les applications de langues sont souvent répétitives et ne ciblent pas réellement les besoins cognitifs de l’apprenant.  
La motivation diminue rapidement.

## Description courte

Une série de mini-jeux basés sur les intentions communicatives.  
Le jeu s’adapte en temps réel au profil sémantique de l’apprenant.

## Fonctionnement

1. Le système identifie les intentions les moins maîtrisées par l’utilisateur.
2. Il propose un défi parmi :
   - défi “mots interdits”,
   - reformulation intelligente,
   - compléter une intention (ex : exprimer un doute),
   - duel de synonymes,
   - défi chronométré.
3. L’utilisateur gagne : XP, badges, niveaux sémantiques.
4. Les résultats alimentent le profil “Digital Self”.

## Valeur pour l’utilisateur

- Motivation accrue via des défis courts et stimulants.
- Renforcement ciblé des intentions faibles.
- Apprentissage naturel par répétition variée.
- Tableau de bord visuel montrant la progression réelle.

## Innovation clé

- Gamification centrée sur les **intentions sémantiques**, et non sur la grammaire.
- Défis personnalisés selon le profil sémantique de chaque utilisateur.
- Boucle d’apprentissage continue : écriture → correction → jeu → évolution du profil.

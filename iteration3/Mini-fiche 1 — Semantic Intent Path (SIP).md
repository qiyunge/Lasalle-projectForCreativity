# Mini-fiche 1 — Semantic Intent Path (SIP)

## Nom de la solution

**Semantic Intent Path (SIP)** – Modèle de navigation sémantique pour l’apprentissage des langues.

## Problème ciblé

Les apprenants reçoivent des corrections, mais ne comprennent pas la logique sémantique derrière les phrases.  
L’apprentissage manque de repères, ce qui réduit la mémorisation et la progression.

## Description courte

Le SIP est un modèle qui mappe chaque phrase vers une intention sémantique (ex : exprimer un avis, comparer, raconter un souvenir, exprimer un doute, etc.).  
L’apprenant suit une carte de progression fondée sur des intentions proches, ce qui crée un apprentissage naturel et cohérent.

## Fonctionnement

1. L’utilisateur écrit une phrase.
2. L’IA détecte l’intention sémantique via un classifieur.
3. Le système affiche :
   - des expressions proches,
   - des structures alternatives,
   - un chemin de progression (“intent path”).
4. L’utilisateur navigue entre des intentions voisines pour enrichir son expression.

## Valeur pour l’utilisateur

- Compréhension profonde des **fonctions communicatives**.
- Progression naturelle d’un contexte maîtrisé vers un contexte plus complexe.
- Mémorisation accrue grâce aux **liens sémantiques**.
- Apprentissage personnalisé selon les intentions dominantes du profil de l’utilisateur.

## Innovation clé

- Premier modèle d’apprentissage basé sur une **topologie sémantique**.
- Passage de “corriger mes erreurs” à “comprendre ce que je voulais exprimer”.
- Base solide pour un **profil sémantique évolutif (Digital Self)**.

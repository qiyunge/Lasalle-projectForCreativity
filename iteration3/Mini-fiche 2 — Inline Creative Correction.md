# Mini-fiche 2 — Inline Creative Correction

## Nom de la solution

**Inline Creative Correction** – Correction en ligne enrichie et interactive.

## Problème ciblé

Les outils actuels corrigent les erreurs mais n’expliquent pas clairement le raisonnement.  
L’apprenant reste passif et ne développe pas d’autonomie linguistique.

## Description courte

Le système affiche la correction directement entre les lignes du texte, avec :

- l’erreur mise en évidence,
- la suggestion,
- une explication courte,
- un bouton “✔ J’accepte / ✖ Je refuse” qui influence le modèle d’apprentissage.

## Fonctionnement

1. L’utilisateur écrit un texte libre.
2. L’IA identifie les erreurs (syntaxe, grammaire, sémantique).
3. Chaque erreur génère un bloc inline contenant :
   - la correction proposée,
   - la justification,
   - une alternative plus naturelle ou créative.
4. Le système enregistre le comportement de l’utilisateur pour affiner son profil linguistique.

## Valeur pour l’utilisateur

- Compréhension réelle de **pourquoi** une phrase est correcte ou non.
- Amélioration immédiate du style et de la précision.
- L’apprenant devient actif : il choisit, compare, expérimente.
- Mémoire d’apprentissage basée sur les actions réelles de l’utilisateur.

## Innovation clé

- Première intégration d’une correction **pédagogique + comportementale**.
- Mélange d’explication linguistique + recommandation stylistique.
- Construction progressive d’un **profil d’apprentissage intelligent**.
